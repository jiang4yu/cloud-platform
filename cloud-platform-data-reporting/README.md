# FTP报文清单（基于细则附录B/C）

说明：所有FTP报文均遵循“文件头+文件体”结构，字段分隔符为英文“;”，字段结束符为“~”，数据组结尾符为“||”，编码格式UTF-8，文件名遵循固定命名规则：**煤矿编码_矿井分类_数据类型常量_数据上传时间-随机4位正整数.txt**（如140421007866_1_AZWZJC_20240629112424_1234.txt）。
